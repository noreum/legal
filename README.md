# Noreum Legal Repository

This repository contains the **official legal and licensing framework** for the Noreum Network.

It defines ownership, usage rights, and licensing terms for all visual, creative, and technical assets associated with the Noreum ecosystem.

---

##  License Overview

| Asset Type | License | Description |
|-------------|----------|--------------|
| **Visual & Creative Assets** (images, artwork, branding, whitepapers, text) | [CC BY 4.0](https://creativecommons.org/licenses/by/4.0/) | Free to use, share, or adapt **with attribution** to Noreum Network. |
| **Source Code & Scripts** | [MIT License](https://opensource.org/licenses/MIT) | Open-source, free for reuse and modification under permissive terms. |

---

##  Attribution Format

When reusing or referencing Noreum materials, include one of the following attribution lines:

> “© 2025 Noreum Network — used under CC BY 4.0 License.”

or for code:

> “Portions © 2025 Noreum Network — Licensed under MIT License.”

---

##  Legal Integrity

All files, manifests, and documents in this repository are:
- Authored and owned by the **Noreum Foundation**
- **Cryptographically signed** using the official cold GPG key
- Version-controlled for historical and legal traceability
